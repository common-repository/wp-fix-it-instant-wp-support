WP Support just got easier and faster. Submit WP Support Tickets directly from your administration area of your website.

== Description ==

This plugin makes getting WP support easier and faster. Use this plugin for submitting a support ticket to WP Fix It for Instant Support. Now if you are working on your site and notice a problem that you can not figure out yourself, simply submit a support ticket right from within you administrative area. Your issue is automatically entered into the support ticket system at [WPfixIt.com](http://wpfixit.com).

<strong>There is NO EASIER or FASTER way to get INSTANT WP SUPPORT!</strong>

== Installation ==

= Install from WP Dashboard =
  * Log into WP dashboard then click **Plugins** > **Add new** > Then under the title "Install Plugins" click **Upload** > **choose the zip** > **Activate the plugin!**

= Install from FTP =
  * Extract the zip file and drop the contents in the wp-content/plugins/ directory of your WP installation and then activate the Plugin from Plugins page. 

= THAT IS IT: You're done! =
